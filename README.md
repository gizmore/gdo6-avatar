# gdo6-avatar

Avatar module for gdo6.


## Features

- Set default avatars for guests, members, males and females.


### Install

    cd GDO
    git clone https://github.com/gizmore/gdo6-avatar Avatar

Then install the module via gdo6 admin panel.


### Dependencies

    gdo6-file (is in core atm)
    gdo6-user (is in core atm)
